> # Arithmetic Operators In Javascript:

Arithmetic operators perform arithmetic on numbers (literals or variables).

|    Operator	|    Description            |
| :---:         |    :---:                  |
|    +	        |    Addition               |
|    -	        |    Subtraction            |
|    *	        |    Multiplication         |
|    **	        |    Exponentiation (ES2016)|
|    /	        |    Division               |
|    %	        |    Modulus (Remainder)    |
|    ++	        |    Increment              |
|    --	        |    Decrement              |

Here,
x ** y produces the same result as Math.pow(x,y).
