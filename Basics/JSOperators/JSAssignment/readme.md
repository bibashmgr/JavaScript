> # Assigment Opeartors In Javascript:

Assignment operators assign values to variables.

|Operator	|    Example	    |     Same As       |
| :---:     |   :---:           |   :---:           |
|`=`        |    x = y	        |    x = y          |
|`+=`	    |    x += y	        |    x = x + y      |
|`-=`	    |    x -= y	        |    x = x - y      |
|`*=`	    |    x *= y	        |    x = x * y      |
|`%=`	    |    x %= y	        |    x = x % y      |
|`<<=`	    |    x <<= y	    |    x = x << y     |
|`>>=`	    |    x >>= y	    |    x = x >> y     |
|`>>>=`	    |    x >>>= y	    |    x = x >>> y    |
|`&=`	    |    x &= y	        |    x = x & y      |
|`^=`	    |    x ^= y	        |    x = x ^ y      |
|`|=`	    |    x |= y	        |    x = x | y      |
|`**=`	    |    x **= y	    |    x = x ** y     |
